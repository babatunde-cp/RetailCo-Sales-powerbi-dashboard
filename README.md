# RetailCo Sales Dashboard 2022-2024 (Power BI)

## Overview
This project analyzes retail sales performance using Power BI.

The dashboard provides executive-level insights into:
- Total Revenue
- Profit Margin
- Discount Rate
- Average Transaction Value
- Store Performance Ranking
- Regional Sales Trends

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling

## Key Features
• KPI cards with performance indicators  
• Store ranking and performance analysis  
• Regional sales distribution  
• Sales trend analysis over time  
• Executive summary page
